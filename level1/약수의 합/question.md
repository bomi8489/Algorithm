## 문제 설명

> 정수 n을 입력받아 n의 약수를 모두 더한 값을 리턴하는 함수, solution을 완성해주세요.

<br>

## 제한 사항

- `n`은 0 이상 3000이하인 정수입니다.

<br>

## 입출력 예

|n	|return|
|:-:|:------:|
|12	|28|
|5	|6|

입출력 예 설명

입출력 예 #1
12의 약수는 1, 2, 3, 4, 6, 12입니다. 이를 모두 더하면 28입니다.

입출력 예 #2
5의 약수는 1, 5입니다. 이를 모두 더하면 6입니다.

<br> 

## 풀이

- n을 1부터 n까지의 수 i로 나누어 나머지가 0이 되는 `n % i === 0`이 되는 모든 i의 합을 구한다. 

```
function solution(n) {
    var answer = 0;
    for(let i=1; i<=n; i++) {
        if(n % i === 0) answer += i;
    }
    return answer;
}
```
